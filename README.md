# CommandCenters
Giải pháp AI tự động kiểm tra và phát hiện các sự cố (hiện đang xây dựng)

    Phần mềm phân tích sự cố hành lang do cây cối gây ra: tính năng, kịch bản kiểm tra
    Phần mềm phân tích các hỏng hóc tiềm ẩn: tính năng, kịch bản kiểm tra
    Phần mềm phát hiện các thành phần của lưới điện: tính năng, kịch bản kiểm tra

Các tính năng chưa xây dựng

    Tự động gửi cảnh báo sự cố đến các bên liên quan
    Tự động định vị và điều phối ứng cứu sự cố lưới điện, hành lang lưới điện
    Kiểm soát kết quả thực hiện ứng cứu sự cố lưới điện, hành lang lưới điện
    Lưu trữ dữ liệu, tìm kiếm và quản lý thông tin
    Lập báo cáo, thống kê
    Dữ liệu hệ thống được mã hóa và đồng bộ trên cloud

## Installing

    git clone https://github.com/VNISTR-D/CommandCenters.git

## Usage
    tạo Virtual Enviroment bên ngoài project 
    ví dụ: VirtualEnvironment
    và bật chế độ môi trường ảo:

    - Window:
        VirtualEnvironment\Scripts\activate.bat
    - Ubuntu:
        source VirtualEnvironment/bin/activate

    cd CommandCenters

    pip install -r requirements.txt
